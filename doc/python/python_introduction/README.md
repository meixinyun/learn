#### Hello World!

```python

print "Hello World!"
```

You can either:

- `./code/hello.py` or `python code/hello.py`
- `cd code/` and `sudo chmod +x hello.py` and `./hello.py`
- `python` and `print "Hello World!"` in the intepreter
